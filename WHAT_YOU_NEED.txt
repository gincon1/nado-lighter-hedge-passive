║                      3 步上传流程                           ║
╚════════════════════════════════════════════════════════════╝

步骤 1: 在 GitHub 创建仓库
   访问: https://github.com/new
   仓库名: nado-lighter-hedge
   选择: Private (推荐)
   不勾选: "Initialize with README"
   点击: Create repository

步骤 2: 解压项目
   tar -xzf nado-lighter-hedge.tar.gz
   cd nado-lighter-hedge

步骤 3: 运行上传脚本
   ./github-upload.sh
   
   输入你的 GitHub 用户名
   完成！
